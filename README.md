### Marketplace Data Matching System

A comprehensive system for matching categories, attributes, and list of values (LOV) across different e-commerce marketplaces.

**Features:**
- Intelligent category matching using hierarchical path comparison
- Attribute mapping with configurable similarity thresholds
- List of Values (LOV) normalization and matching
- Detailed JSON and text-based reports
- Example datasets from Amazon and Flipkart marketplaces

**Quick Start:**
```bash
cd marketplace_matching/scripts
python3 marketplace_matcher.py
```

See the [full documentation](marketplace_matching/README.md) for more details.
